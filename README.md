# DEM LiDAR

This repository contains models and notebooks for training LiDAR / DG-VDSR models.

## Dataset
The dataset for this project is stored in Google Drive, as training is intended to be run in a Google Colab environment. Please ensure that you have mounted Google Drive and the dataset paths match the directories expected by the notebooks.

## Setup
Install the required dependencies using:
```bash
pip install -r requirements.txt
```
